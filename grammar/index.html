1. 동사의 3가지 분류

일본어 동사는 활용 방식에 따라 3그룹으로 나뉘어요. 이것만 정확히 구분하면 모든 변형이 쉬워집니다.

그룹 특징 구분 방법 예시
1그룹
5단동사
(う동사)		 끝이 う단으로 끝남
(う, く, ぐ, す, つ, ぬ, ぶ, む, る)		 る로 안 끝나거나,
る 앞이 あ·う·お단		 書かく (쓰다)
話はなす (말하다)
帰かえる (돌아가다) ⚠️
2그룹
1단동사
(る동사)		 끝이 る로 끝남
る 앞이 い단 · え단		 る 앞의 소리 확인 見みる (보다)
食たべる (먹다)
起おきる (일어나다)
3그룹
불규칙		 딱 2개만 존재 외워버리기 する (하다)
来くる (오다)
⚠️ 함정 동사 (예외 1그룹)

형태는 2그룹처럼 보이지만 1그룹인 동사들. 반드시 암기!

  • 帰かえる (돌아가다) · 入はいる (들어가다) · 走はしる (달리다)
  • 知しる (알다) · 切きる (자르다) · 要いる (필요하다)
  • 喋しゃべる (수다 떨다) · 滑すべる (미끄러지다) · 散ちる (떨어지다)

2. 동사 활용 총정리 표

각 그룹별로 어떻게 변형되는지 한눈에 비교하세요. 書かく (쓰다, 1그룹), 食たべる (먹다, 2그룹), する (하다, 3그룹) 으로 예시를 통일했습니다.

활용형 의미 / 쓰임 1그룹 書かく 2그룹 食たべる 3그룹 する
사전형
(辞書形)		 기본형, 반말 書かく 食たべる する
ます형 정중형 (~합니다) 書かきます 食たべます します
ない형 부정 (~하지 않다) 書かかない 食たべない しない
て형 연결 (~하고/해서) 書かいて 食たべて して
た형 과거 (~했다) 書かいた 食たべた した
의지형
(意向形)		 ~하자/하려 하다 書かこう 食たべよう しよう
명령형 ~해! (거친 명령) 書かけ 食たべろ しろ/せよ
가정형
(ば형)		 ~하면 書かけば 食たべれば すれば
가능형 ~할 수 있다 書かける 食たべられる できる
수동형
(受け身)		 ~당하다/받다 書かかれる 食たべられる される
사역형
(使役)		 ~시키다/하게 하다 書かかせる 食たべさせる させる
사역수동
(使役受け身)		 (억지로) ~하게 됨 書かかされる
書かかせられる		 食たべさせられる させられる

1그룹 활용 규칙 (う단 → 다른 단)

만들 형태바꾸는 법예: 書く (か く)
ます형う단 → い단 + ます書きます
ない형う단 → あ단 + ない
(단, う로 끝나면 あ가 아니라 わ)		書かない
買う → 買わない
의지형う단 → お단 + う書こう
가정형う단 → え단 + ば書けば
가능형う단 → え단 + る書ける
수동형う단 → あ단 + れる書かれる
사역형う단 → あ단 + せる書かせる
명령형う단 → え단書け

2그룹 활용 규칙 (る 떼고 연결)

훨씬 쉬워요! る를 떼고 필요한 어미만 붙이면 끝.

  • 食べる → 食べ + ます / ない / て / た / よう / れば / られる
  • 食べ + られる → 가능형 & 수동형이 같음 (문맥으로 구분)
  • 食べ + させる → 사역형
  • 명령형: 食べ + ろ
💡 2그룹 가능형: ら 뺀 「食べれる」?

요즘 회화에선 「食べられる」 대신 「食べれる」로 줄여 말하는 "ら抜き言葉"가 흔해요. 시험에선 공식 형태인 られる를 쓰세요.

3그룹 (불규칙 — 통째로 암기)

활용형する来くる
ます형します来きます
ない형しない来こない
て형して来きて
의지형しよう来こよう
명령형しろ / せよ来こい
가능형できる来こられる
수동형される来こられる
사역형させる来こさせる

3. て형·た형 만들기 (1그룹 규칙이 핵심!)

て형과 た형은 만드는 법이 같아요 (て → た 로만 바꿈). 1그룹 동사는 끝자음에 따라 변해서 외워야 해요.

~う, ~つ, ~る ~む, ~ぶ, ~ぬ ~く ~ぐ ~す
→ ～って → ～んで → ～いて → ～いで → ～して
買かう → 買って
待まつ → 待って
帰かえる → 帰って 		飲のむ → 飲んで
呼よぶ → 呼んで
死しぬ → 死んで 		書かく → 書いて
※예외
行いく → 行って 		泳およぐ → 泳いで
急いそぐ → 急いで 		話はなす → 話して
出だす → 出して
🎵 외우는 노래

「うつる → って / むぶぬ → んで / く → いて / ぐ → いで / す → して」
리듬 타서 10번만 소리 내서 읽으면 평생 안 잊어요!

て형의 주요 용법

① ～ている

V-て + いる

진행 / 상태 (~하고 있다 / ~해 있다)

  • 今いま本ほんを読よんでいる。 지금 책을 읽고 있다.
  • 窓まどが開あいている。 창문이 열려 있다. (상태)

② ～てください

V-て + ください

정중한 부탁 (~해 주세요)

  • ちょっと待まってください。 잠시만 기다려 주세요.

③ ～てもいい / ～てはいけない

V-て + もいい / はいけない

허가 (~해도 된다) / 금지 (~하면 안 된다)

  • ここで写真しゃしんを撮とってもいいですか。 여기서 사진을 찍어도 되나요?
  • ここでタバコを吸すってはいけません。 여기서 담배를 피우면 안 됩니다.

④ ～てから / ～たあとで

V-て + から / V-た + あとで

순서 (~하고 나서 / ~한 후에)

  • 手てを洗あらってから食たべます。 손을 씻고 나서 먹습니다.

⑤ ～てしまう / ～ちゃう(구어)

V-て + しまう

완료 강조 / 아쉬움·후회 (~해 버리다)

  • 財布さいふを忘わすれてしまった。 지갑을 잊어 버렸다.

⑥ ～ておく / ～とく(구어)

V-て + おく

미리 ~해 두다

  • 会議かいぎの前まえに資料しりょうを読よんでおきます。 회의 전에 자료를 읽어 두겠습니다.

⑦ ～てみる

V-て + みる

시도 (~해 보다)

  • この料理りょうり、食たべてみてください。 이 요리, 먹어 봐 주세요.

4. 수동형 (受け身み) — 당하다 / 받다

만드는 법

그룹규칙예시
1그룹う단 → あ단 + れる叱しかる → 叱られる (혼나다)
2그룹る → られる見る → 見られる (보여지다)
3그룹する → される / 来る → 来(こ)られる注意する → 注意される (주의 받다)

3가지 용법

① 직접 수동 — 누가 내게 뭘 했다

A は B に + V-れる/られる

A는 B에게 ~당하다/받다

  • 私わたしは先生せんせいに褒ほめられた。 나는 선생님께 칭찬 받았다.
  • 犬いぬに噛かまれた。 개에게 물렸다.

② 간접 수동 (피해 수동) — 일본어 특유!

A は B に + (C を) + V-れる/られる

A는 B가 (C를) ~해서 피해를 입다 / 난감하다

  • 雨あめに降ふられた。 비를 맞아서 곤란했다. (비가 내려서 나에게 피해)
  • 隣となりの人ひとにタバコを吸すわれた。 옆 사람이 담배를 피워서 (내가) 괴로웠다.
  • 友達ともだちに来こられて、勉強べんきょうできなかった。 친구가 와서 (방해돼서) 공부 못했다.

③ 무생물 수동 — 사물이 주어

N が + V-れる/られる

신문·보도·객관 서술에 많이 사용

  • この本ほんは多おおくの人ひとに読よまれている。 이 책은 많은 사람들에게 읽히고 있다.
  • オリンピックは東京とうきょうで開ひらかれた。 올림픽은 도쿄에서 개최되었다.
⚠️ 조사 주의

수동문에선 행위자 뒤에 「に」 (혹은 「によって」 - 공식/창작자). 한국어 "~에게" 감각과 비슷해요.

5. 사역형 (使役えき) — 시키다 / 하게 하다

만드는 법

그룹규칙예시
1그룹う단 → あ단 + せる書かく → 書かせる
2그룹る → させる食たべる → 食べさせる
3그룹する → させる / 来る → 来こさせる勉強べんきょうする → 勉強させる

두 가지 뉘앙스: 강제 vs 허용

① 강제 — 억지로 시키다

A は B を / に + V-せる/させる
  • 母ははは弟おとうとに野菜やさいを食たべさせた。 엄마는 남동생에게 야채를 먹게 했다.
  • 先生せんせいは学生がくせいを立たたせた。 선생님은 학생을 세웠다.

② 허용 — ~하게 놔 두다 / 시켜 주다

  • 子こどもを好すきなだけ遊あそばせた。 아이가 원하는 만큼 놀게 해 주었다.

③ 정중한 부탁: 「～させてください」

V-せて/させて + ください

"~하게 해 주세요" = 자기가 하겠다고 양해를 구하는 정중 표현

  • ちょっと考かんがえさせてください。 잠시 생각하게 해 주세요 (= 생각할 시간을 주세요).
  • その仕事しごと、私わたしにやらせてください。 그 일, 제가 하게 해 주세요.
🔑 を vs に 구분 포인트
  • 자동사 사역 → を (학생을 세우다)
  • 타동사 사역 → に (동생에게 야채를 먹이다) — 대상과 혼동 방지

6. 사역수동 (使役受うけ身) — 억지로 ~하게 되다

사역형 + 수동형의 결합. "남이 시켜서 (내가) 억지로 ~했다" 는 뉘앙스. 불만·피해 감정이 담김.

그룹규칙예시
1그룹 う단 → あ단 + される
(정식: せられる → 축약)		 飲のむ → 飲まされる
(정식: 飲ませられる)
2그룹 る → させられる 食べる → 食べさせられる
3그룹 する → させられる
来くる → 来こさせられる		 勉強する → 勉強させられる
⚠️ 1그룹 축약 주의

1그룹은 회화에서 ～される 로 축약돼요.
단, ~す로 끝나는 동사 (話す, 出す 등)는 축약형을 쓰지 않고 ～させられる 를 사용.
예: 話す → 話させられる (○) / 話さされる (×)

예문

  • 子こどもの時とき、母ははに野菜やさいを食たべさせられた。 어릴 때 엄마 때문에 (억지로) 야채를 먹어야 했다.
  • 友達ともだちに一いち時間じかん待またされた。 친구 때문에 (억지로) 1시간이나 기다려야 했다.
  • 会社かいしゃで毎日まいにち残業ざんぎょうさせられている。 회사에서 매일 (억지로) 야근하고 있다.
🎯 구분 요약
飲ませる (사역)내가 남에게 마시게 시키다
飲まれる (수동)(내 것을) 누가 마셔 버리다
飲まされる (사역수동)남이 시켜서 (내가) 억지로 마시다

7. 가능형·의지형·명령형

가능형 (可能のう) — ~할 수 있다

그룹규칙예시
1그룹う단 → え단 + る書く → 書ける, 飲む → 飲める
2그룹る → られる食べる → 食べられる
3그룹する → できる
来る → 来こられる		運転する → 運転できる
📝 가능형 주의
  • 조사 변화: を → が (にほんごが話せる)
  • 「見る · 聞く」 는 「見える · 聞こえる」 도 씀 (자연스럽게 보이고/들리고)

의지형 (意向こう形) — ~하자 / ~하려고

그룹규칙예시
1그룹う단 → お단 + う行く → 行こう
2그룹る → よう食べる → 食べよう
3그룹する → しよう / 来る → 来こよう

주요 용법

  • 一緒いっしょに行いこう！같이 가자!
  • 留学りゅうがくしようと思おもっている。유학하려고 생각 중이다. (～ようと思う)
  • 寝ねようとした時とき、電話でんわが鳴なった。자려 할 때 전화가 울렸다. (～ようとする)

명령형 — ~해! (강하게)

그룹규칙예시
1그룹う단 → え단書く → 書け, 走る → 走れ
2그룹る → ろ食べる → 食べろ
3그룹する → しろ / せよ
来る → 来こい
🗣️ 명령형 사용 주의

명령형은 매우 거칠어요. 실생활에선 ① 부모→자식 ② 군대·스포츠 ③ 표지판(「止とまれ」 멈춰!) ④ 격한 감정 표현 정도에만 씀. 대신 일상에선 ～てください / ～なさい 사용.

8. 조건형 4가지 (ば / たら / なら / と)

한국어로는 다 "~하면" 이지만, 일본어는 미묘하게 다르게 씁니다.

형태핵심 뉘앙스예시
～と 항상 그렇게 된다 (자연법칙·습관·길안내) 春はるになると桜さくらが咲さく。
봄이 되면 벚꽃이 핀다.
～ば 가정·조건의 기본. 후문에 과거/명령 X 安やすければ買かいます。
싸면 살게요.
～たら 가장 자유로움. 일회성·구어체 雨あめが降ふったら、家いえにいます。
비가 오면 집에 있을게요.
～なら 주제 제시형 "~라면" (상대 발화를 받아) 温泉おんせんなら箱根はこねがいいですよ。
온천이라면 하코네가 좋아요.
🧠 쉬운 선택 팁
  1. 자연법칙·100% 결과 → と
  2. 일상 대화, 일회성 → たら (가장 무난)
  3. 격언·속담, 가능조건 → ば
  4. 상대가 말한 주제에 대한 조언 → なら

9. 형용사 활용 (い형용사 · な형용사)

い형용사 (끝이 い로 끝남)

예: 高たかい (비싸다/높다)

형태규칙예
기본い高い
부정い → くない高くない
과거い → かった高かった
과거부정い → くなかった高くなかった
て형 (연결)い → くて高くて (비싸고)
부사화い → く高く (높이)
명사 수식い + 명사高い本ほん
⚠️ 예외: いい (좋다)

「いい」 는 활용할 때 よい 의 형태로 변해요.
いい → よくない / よかった / よくなかった / よくて

🔥 헷갈리는 は vs が
  • は: 이미 아는 주제, 대조, 문장 전체의 흐름
  • が: 새 정보, 답 ("누가/무엇이?"에 대한 답), 존재문(います/あります)
  • 예: 象ぞうは鼻はなが長ながい。 (코끼리는 코가 길다 → 주제: 코끼리, 주어: 코)

N5 11. JLPT N5 핵심 문법

～は～です / ～じゃありません

N は N です (~은/는 ~입니다)
  • 私わたしは韓国人かんこくじんです。저는 한국인입니다.
  • 学生がくせいじゃありません。학생이 아닙니다.

～があります / ～がいます

무생물: あります / 생물: います
  • 机つくえの上うえに本ほんがあります。책상 위에 책이 있습니다.
  • 公園こうえんに犬いぬがいます。공원에 개가 있습니다.

～たい (~하고 싶다)

V-ます형 + たい
  • 水みずが飲のみたいです。물을 마시고 싶어요.

～ませんか / ～ましょう

권유 (~하지 않을래요? / ~합시다)
  • コーヒーを飲のみませんか。커피 마시지 않을래요?
  • 一緒いっしょに行いきましょう。같이 갑시다.

～から (이유) / ～前に / ～後で

  • 疲つかれたから休やすみます。피곤해서 쉽니다.
  • 食事しょくじの前まえに手てを洗あらう。식사 전에 손을 씻는다.

～ながら (동시 동작)

V-ます형 + ながら
  • 音楽おんがくを聞ききながら勉強べんきょうする。음악을 들으면서 공부한다.

～が上手じょうずだ / ～が下手へただ / ～が好すきだ

감정·능력은 조사 が!

  • 彼かれは歌うたが上手じょうずです。그는 노래를 잘합니다.

～でしょう (추측·확인)

  • 明日あしたは雨あめでしょう。내일은 비가 올 겁니다.

N4 12. JLPT N4 핵심 문법

～ようになる / ～なくなる

변화 (~하게 되다 / ~하지 않게 되다)
  • 日本語にほんごが話はなせるようになった。일본어를 말할 수 있게 되었다.
  • 肉にくを食たべなくなった。고기를 먹지 않게 되었다.

～ようにする (의식적으로 노력)

  • 毎日まいにち運動うんどうするようにしている。매일 운동하려고 하고 있다.

～と思おもう / ～と言いう

인용 (~라고 생각한다 / ~라고 한다)
  • これはいいと思おもいます。이건 좋다고 생각합니다.
  • 田中たなかさんは「行いかない」と言いった。다나카 씨는 "안 간다"고 했다.

～ば～ほど (~하면 ~할수록)

  • 練習れんしゅうすればするほど上手じょうずになる。연습하면 할수록 잘하게 된다.

～なければならない / ～なきゃ(구어)

의무 (~해야 한다)
  • 薬くすりを飲のまなければなりません。약을 먹어야 합니다.
  • 早はやく帰かえらなきゃ。빨리 돌아가야 해.

～なくてもいい (~하지 않아도 된다)

  • 明日あしたは来こなくてもいいです。내일은 안 와도 됩니다.

～つもりだ / ～予定ていだ

의도·계획 (~할 생각이다 / ~할 예정이다)
  • 来年らいねん結婚けっこんするつもりです。내년에 결혼할 생각입니다.

～かもしれない / ～はずだ

  • 雨あめが降ふるかもしれません。비가 올지도 모릅니다. (50%)
  • 彼かれはもう着ついているはずです。그는 이미 도착했을 겁니다. (근거 있는 확신)

～そうだ (양태·전문)

  • おいしそうですね！맛있어 보이네요! (양태: 보고 추측, い형은 い 떼고)
  • 明日あしたは雨あめが降ふるそうです。내일은 비가 온다고 해요. (전문: 들은 정보, 보통형+そうだ)

～やすい / ～にくい

V-ます형 + やすい(쉽다) / にくい(어렵다)
  • この本ほんは読よみやすい。이 책은 읽기 쉽다.
  • この靴くつは歩あるきにくい。이 신발은 걷기 불편하다.

N3 13. JLPT N3 핵심 문법

N3는 표현이 훨씬 풍부해져요. 시험뿐 아니라 드라마·애니에서도 자주 만나는 표현들입니다.

～わけだ / ～わけではない

わけだ: 그래서 ~인 거구나 (납득) / わけではない: ~인 것은 아니다 (부분 부정)

  • 10年じゅうねん住すんでいるから、日本語にほんごが上手じょうずなわけだ。10년 살고 있으니까 일본어를 잘하는 거구나.
  • 嫌きらいなわけではない。싫은 것은 아니다.

～ばかり (~뿐)

① ～てばかりいる: ~만 하고 있다 / ② ～たばかり: 막 ~했다

  • 弟おとうとはゲームをしてばかりいる。동생은 게임만 하고 있다.
  • ついさっき着ついたばかりです。방금 막 도착했습니다.

～ところだ / ～たところだ / ～ているところだ

시점 표현 (막 ~하려던 참 / 막 ~한 참 / ~하고 있는 중)

  • これから食たべるところです。이제 막 먹으려던 참이에요.
  • もう食たべたところです。방금 막 먹었어요.
  • いま食たべているところです。지금 먹고 있는 중이에요.

～ように / ～ないように

목적 (~하도록 / ~하지 않도록)

  • 忘わすれないようにメモする。잊지 않도록 메모한다.

～ために (~를 위해 / ~때문에)

목적·원인 (동사 사전형 → 목적 / 명사+の → 목적 / た형·원인)

  • 健康けんこうのために運動うんどうする。건강을 위해 운동한다.
  • 台風たいふうのため、電車でんしゃが止とまった。태풍 때문에 전철이 멈췄다.

～おかげで / ～せいで

긍정 원인 (~덕분에) / 부정 원인 (~탓에)

  • 先生せんせいのおかげで合格ごうかくできた。선생님 덕분에 합격할 수 있었다.
  • 雨あめのせいで遅おくれた。비 탓에 늦었다.

～について / ～に関かんして

주제 제시 (~에 대해서 / 좀 더 격식)

  • 環境問題かんきょうもんだいについて話はなす。환경 문제에 대해 이야기한다.

～にとって (~에게 있어서)

  • 私わたしにとって家族かぞくが一番いちばん大切たいせつです。나에게 가족이 가장 소중합니다.

～によって

수단·원인·주체·차이 (~에 의해 / ~에 따라)

  • この小説しょうせつは村上春樹むらかみはるきによって書かかれた。이 소설은 무라카미 하루키에 의해 쓰였다.
  • 国くにによって文化ぶんかが違ちがう。나라마다 문화가 다르다.

～代かわりに

~대신에

  • 父ちちの代かわりに私わたしが行いきます。아버지 대신 제가 가겠습니다.

～通とおりに / ～通どおり

~대로

  • 説明書せつめいしょのとおりに作つくる。설명서대로 만든다.

～間あいだ / ～間あいだに

~동안 (계속) / ~하는 사이에 (그 동안에 어떤 일이)

  • 赤あかちゃんが寝ねている間あいだ本ほんを読よんだ。아기가 자는 동안 계속 책을 읽었다.
  • 赤あかちゃんが寝ねている間あいだに買かい物ものに行いった。아기가 자는 사이에 장보러 갔다 왔다.

～際さいに / ～最中さいちゅうに

~할 때 (격식) / 한창 ~하는 중에

  • ご利用りようの際さいはご注意ちゅういください。이용 시 주의해 주세요.
  • 会議かいぎの最中さいちゅうに電話でんわが鳴なった。회의가 한창인 중에 전화가 울렸다.

～としても / ～にしても

~라고 해도 (가정의 양보)

  • どんなに忙いそがしいとしても、連絡れんらくはしてほしい。아무리 바쁘다 해도 연락은 해 줬으면.

～ば良よかった / ～なければ良よかった

후회 (~했으면 좋았을걸 / ~하지 않았으면 좋았을걸)

  • もっと早はやく出でればよかった。좀 더 빨리 나올 걸 그랬다.
  • あんなことを言いわなければよかった。그런 말 안 했으면 좋았을걸.

～ものだ / ～ものではない

일반적 진리·상식·당연함

  • 子こどもは元気げんきなものだ。아이는 (원래) 건강한 법이다.
  • そんなことを言いうものではない。그런 말을 하는 게 아니다.

～がる (감정의 객관적 표현)

남의 감정은 ～がる로 (い형 い떼고 / な형 な떼고 + がる)

  • 子こどもが寒さむがっている。아이가 추워한다.
  • 妹いもうとはその本ほんを欲ほしがっている。여동생이 그 책을 갖고 싶어한다.

～らしい (~인 것 같다 / ~답다)

  • 彼かれはもう帰かえったらしい。그는 이미 돌아간 것 같다. (객관적 추측)
  • 彼かれは男おとこらしい人ひとだ。그는 남자다운 사람이다.

～てほしい / ～てもらいたい

남에게 ~해 주기 바란다

  • 手伝てつだってほしい。도와줬으면 좋겠어.
